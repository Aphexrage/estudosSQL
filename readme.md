<h2 align="center">🛢️ Repositório de Estudos SQL</h2>

Este repositório contém meus estudos e treinamentos com a linguagem SQL. Durante o processo de aprendizado, utilizei recursos da **Microsoft Azure**, onde configurei um grupo de recursos e um **SQL Server**. O banco de dados utilizado foi o **AdventureWorks**, fornecido pela própria Microsoft para fins educacionais.

### 🔧 Ferramentas Utilizadas

- Microsoft Azure,
- DBeaver,  
- MySQL Workbench,
- SQL Server Management Studio,  

### 📌 Objetivo

O objetivo deste repositório é registrar e compartilhar meu progresso e práticas com SQL. Estou revisando os principais conceitos para futuramente construir projetos mais avançados.
